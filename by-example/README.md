# Rust by Example - Exercises

Working through [Rust by Example](https://doc.rust-lang.org/rust-by-example/).

## Structure

Each topic gets its own folder with small standalone `.rs` files or cargo projects:

```text
by-example/
├── 01-hello/
│   ├── hello.rs
│   └── formatted_print.rs
├── 02-primitives/
│   └── ...
└── ...
```

## Running Standalone Files

```bash
# Compile and run a single file
rustc hello.rs && ./hello

# Or use cargo-script if installed
```
